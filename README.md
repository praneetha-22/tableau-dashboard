# Drug Data Analysis Dashboard

##Project Overview
This project presents a Tableau dashboard created to analyze drug-related data.

##Dataset
The dataset includes:
- Age
- Blood Pressure (BP)
- Cholesterol
- Drug types (drugA, drugB, drugC, drugX, drugY)

##Dashboard Features
- Scatter Plot (Age vs Na_to_K)
- Age Distribution Histogram
- BP vs Drug (Stacked Bar Chart)
- Drug Distribution (Bar Chart)
- Interactive Filters

##Objective
To analyze patterns between patient attributes and drug usage.

##Tool Used
- Tableau
